# edison.qa-show

Prints the QA Markdown file exactly as stored (including YAML frontmatter).

## Usage

```bash
edison qa show <qa_id>
```

## Arguments
- qa_id (required): QA identifier (e.g., 150-wave1-auth-gate-qa)

## When to use

To quickly inspect the raw QA brief as stored on disk
