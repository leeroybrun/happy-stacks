# edison.task-show

Prints the task Markdown file exactly as stored (including YAML frontmatter).

## Usage

```bash
edison task show <task_id>
```

## Arguments
- task_id (required): Task identifier (e.g., 150-wave1-auth-gate)

## When to use

To quickly inspect the raw task document as stored on disk
